## Getting Started

### Prerequisites
Ensure that [Node](https://nodejs.org/en/) and [git](https://git-scm.com/) are installed.

### Run Locally
- Clone the repo:

  ```
  git clone https://github.com/Shmurds28/SovTech-minPortfolio.git
  ```

- Navigate to the project directory

- Install the dependencies:

  ```
   npm install
  ```

- Run Server

  ```
   npm run dev
  ```
